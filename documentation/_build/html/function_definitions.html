dp_ml helper functions¶

Some helper functions for the dielectric property estimation toolbox David Garrett, October 2017

dp_ml.dp_ml_fx.S_to_T(S)¶

Converts S-parameters to T-parameters

dp_ml.dp_ml_fx.T_to_S(T)¶

Converts T-parameters back to S-parameters

dp_ml.dp_ml_fx.get_params_from_filename(fileName)¶

Get permittivity, conductivity, and separation distance from a filename. Example formatted filename: eps20_sig30_50mm.xls For calibration procedures, gives ‘0’ for every param. e.g. Thru.xls

dp_ml.dp_ml_fx.importFromDir(dataPath)¶

Imports data from a given datapath

dp_ml.dp_ml_fx.loadCalibration_WG()¶

Load calibration data for the waveguide

dp_ml.dp_ml_fx.mag2db(mag)¶

Converts a magnitude value to a db representation (using 20*log10(mag))

dp_ml.dp_ml_fx.performCalibration_WG(S_wg, O_wg_T_inv, P_wg_T_inv)¶

Perform antenna calibration with the waveguide O and P matrices

dp_ml.dp_ml_fx.realimag_to_magphase(real, imag)¶

Converts real and imaginary values to a magnitude and phase [rad] value.

dp_ml.hdf5_fx.searchForEntry(dbName, dbHier, indFileName, freqOrTime='/frequency/')¶

Searches through an HDF5 file for existence of an entry

  • dbName is the hdf5 file
  • dbHier is the hierarchical path within the database - e.g. ‘/Data/Simulations/CircWG’
  • indFileName is the name of the recorded file according to your convention. Used to check existence
